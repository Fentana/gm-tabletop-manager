GM Tabletop Manager v0.0.4 by almightynassar. Click here to access the GitHub Repository

WARNING: Cleaning out your cookies will delete ALL stored/edited data! This is because these tools use LocalStorage to save data. If you are clearing out your cookies, chances are it will clear your saved data!

This is not an attempt to replace any licensed material from the publishers. This should be used as a supplement to published materials, and as such it will not include a lot of things that can be found in the official sources. Functions may be provided to allow you to input the required content yourself.

These tools can all run locally on your desktop without the need for a server. This is because it uses JavaScript and the HTML5 LocalStorage feature (think of it as advanced cookies). To download these tools, click on the github link provided above and download the zip file. When you extract the files, double-click on index.html to open the system in your browser!
